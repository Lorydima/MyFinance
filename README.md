# 🎲Features

MyFinance includes:
- Simple CLI
- Command user-friendly
- Simple insertion of incomes or expensives
- Simple wiew of own finances
- View Graph Function

# 💾Downolad MyFinance 
To donwload MyFinance V2.0 follow this link, the software is for **Windows OS, for linux use Wine:**
<a href="https://github.com/Lorydima/MyFinance/releases/download/MyFinanceV2.0/MyFinance_V2.0.zip" download>Download MyFinance V2.0</a>

**For macOS**  
The EXE file is not available.

However, the application can be run from source by executing the `main.py` file,
provided that Python and the required dependencies are installed.

> [!WARNING]
> **For proper program execution, please read the notes below**
> 
>   **Do not delete** the `.json` file in the download folder they are required for the program to function correctly.

# 🔗Clone Repository
```bash
git clone https://github.com/Lorydima/MyFinance.git
```
```bash
pip install -r requirements.txt
```
```bash
run main.py
```
